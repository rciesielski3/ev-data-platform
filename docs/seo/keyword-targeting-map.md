## Notes for Implementation

1. **URL Structure:** All URLs assume existing structure (e.g., `/map`, `/stations`, `/vehicles`). If structure differs, adjust keyword assignments accordingly.

2. **Multilingual Consideration:** Polish and English keywords should be optimized separately but with parallel structure. Use next-intl translations for metadata.

3. **Internal Linking:** Build web of related pages using anchor text matching target keywords (natural, not forced).

4. **Mobile-First:** Ensure all pages are optimized for mobile; many EV charging searches are location-based on-the-go.

5. **Long-Tail Strategy:** After P1 keywords rank, expand with long-tail variations (e.g., "szybkie ładowanie w Warszawie").
